Rust/Cargo Docker
=================

Ready to use development environment with [Rust/Cargo](https://www.rust-lang.org/) + [libtss2](https://github.com/tpm2-software/tpm2-tss), based on Debian Linux.

**Docker Hub page:**  
<https://hub.docker.com/r/danieltrick/rust-tss2-docker>
